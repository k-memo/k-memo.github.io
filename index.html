  • stick

    Overview

    stick is a fighting stick which I made for playing Tekken with my Buddy. I am not a pro by any means but I love fighting games and using a gamepad with a big Joystick and Buttons. I modeled and 3D-printed the shell via Fusion 360 and my beloved Anycubic Chiron. I used Arduino Uno as the logic board, using the UnoJoy Github repo. All in all it was a very fun project and my first time using an Arduino Board!
